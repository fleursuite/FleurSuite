Release Notes
What's new

A full rundown of every change, fix, and improvement in each release.

v1.0.0.2
May 2026 — Latest
☑️
Multi-Select Mode
  • Press Shift once to enter select mode; press again or press Escape to exit
  • Hold Shift and click to select multiple notes without leaving the mode
  • Animated "N Notes Selected" banner appears at the top of the note list
  • Drag any selected note to move the entire selection at once
  • Delete key or trash icon deletes all selected notes with one confirmation
  • Releasing Shift no longer clears the selection
🎨
Redesigned Color Pickers
  • All color pickers (text, highlight, theme) now open and close on click — no more hover timeouts
  • Popups stay open reliably; click anywhere outside to dismiss
  • Settings accent picker rebuilt as an anchored popup matching the toolbar pickers
  • Six new themed palettes: Ink & Paper, Forest & Earth, Ocean & Sky, Rose & Dusk, Violet & Plum, Pastels
  • Removed "Muted & Dusty" and "Vivid" palettes
✅
Working Checklists
  • Checklist circles now respond to clicks correctly
  • Clicking a circle fills it with a checkmark and strikes through the text
  • Checked state persists across saves and note switches
  • Hover highlight on circles for clear affordance
  • Checklist icon in / menu and right-click menu now matches the toolbar icon
🪟
Split View from Sidebar
  • Right-click any note in the sidebar to open it in split view
  • "Open in split view" toggles — right-click again to close that note's split
📋
Sidebar Context Menus
  • Note and folder right-click menus redesigned to match the editor context menu style
  • Wider popup (220px), max height with scrolling, consistent section labels
  • Folder and note name headers now trim cleanly with ellipsis
⚙️
Settings Panel
  • Theme toggle cards now have the same hover and press animation as color theme cards
  • Version number displayed in the settings footer, pulled automatically from the build
🖼️
Rounded Image Shadows
  • Drop shadows now appear correctly on rounded images
  • Fixed by switching from geometry clip to opacity mask, so the shadow renders outside the image bounds
  • Shadow persists correctly after resizing a rounded image
⌨️
Slash Menu Cleanup
  • Hidden commands (like "image" and "Attach Document") no longer appear in the right-click menu
  • Hidden commands now appear properly in the / list when you type their name
  • Removed "linkdocument" duplicate; kept "Attach Document" with proper display name
  • All slash command labels normalized to Title Case
🔧
Bug Fixes
  • Fixed garbled em dashes (â€") in attached document disclaimers and toast messages
  • Fixed attachment card showing corrupted "Missing â€" relink or unlink" text
  • Fixed highlight and text-color popups randomly disappearing mid-interaction
  • Fixed dragging an already-selected note deselecting it before the drag started

📌Important to know

This is a beta, solo-built app — bugs come with the territory. I appreciate your patience while I work through them.

On install, Windows may show an "Unknown Publisher" pop-up. This is normal. Removing it requires an OV code-signing certificate, which I'm saving up for while paying off college. It'll go away eventually! 🙂

If you know of affordable ways to host apps with live updates, I'd genuinely love to hear about it.

FleurSuite will always be free. But if you'd like to support the work (and help me buy that code-signing cert), it means the world. 💐
